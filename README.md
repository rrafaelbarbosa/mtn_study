# mtn_study
App gamificado de estudos sobre modelos de negócio, métricas, KPIs, MVP, SaaS, Marketplace e E-commerce.
O MTN Study Quest é uma aplicação web gamificada criada para apoiar o estudo de conteúdos relacionados a modelos de negócio, métricas, KPIs e estratégias digitais.

A proposta do projeto é transformar um conteúdo teórico em uma experiência mais dinâmica e interativa, utilizando uma trilha de aprendizagem com módulos, quizzes, sistema de XP, progresso e revisão de erros.

Sobre o Projeto
Este projeto foi desenvolvido como um app de estudos em formato de jogo, no qual o usuário avança por módulos temáticos e responde perguntas relacionadas ao conteúdo estudado.

A aplicação possui uma interface visual moderna, com tema escuro, cards de progresso, módulos bloqueados e desbloqueados, simulado geral e acompanhamento de desempenho.

O objetivo é tornar o aprendizado mais envolvente, ajudando o usuário a revisar conceitos importantes de negócios digitais e métricas de forma prática.
