# MTN Study 

O **MTN Study ** é um projeto web interativo desenvolvido com HTML, CSS e JavaScript, criado com o objetivo de transformar conteúdos de estudo em uma experiência mais dinâmica, leve e gamificada.

A proposta do projeto é apresentar uma jornada de aprendizado em formato de jogo, onde o usuário pode interagir com a página, avançar pelas etapas e consumir o conteúdo de forma mais envolvente.

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido como parte de um exercício prático de desenvolvimento web, com foco em aplicar conceitos fundamentais de front-end e criar uma página funcional, visualmente agradável e responsiva.

O principal objetivo é demonstrar:

- Estruturação de páginas com HTML;
- Estilização com CSS;
- Interatividade com JavaScript;
- Organização de arquivos para publicação no GitHub Pages;
- Criação de uma experiência gamificada para estudos.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## 🕹️ Funcionalidades

- Interface interativa em formato de jogo;
- Tela inicial com apresentação do projeto;
- Elementos visuais personalizados;
- Navegação simples e intuitiva;
- Layout adaptado para diferentes tamanhos de tela;
- Publicação online através do GitHub Pages.
